# Install Docker ce and Docker Compose - Bash script
Script to Install Docker CE and compose on Debian/Ubuntu/CentOS

### 1 - Download or clone:
        git clone https://github.com/paulo-amaral/Install-docker-ce-docker-compose.git

### 2 - Set permission:
        chmod +x install

### 3 - Run script:
        ./install


And it's all fine ;)
